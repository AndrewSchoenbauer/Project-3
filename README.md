## Introduction: 
The app is dedicated to trip expenses and offers a simple way to split expenses up while having a secure place to save them. User can log in or make an account and once they‚Äôve done that, can make trips and add expenses to those trips. They also can go back to their saved trips.

## How to use this application:
It is easy-to-use application with multiple features for the user. App mostly use buttons to get around. The application does take user input and store data. User are required to sign up and have an account to access everything on website.
